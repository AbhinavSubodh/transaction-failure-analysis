# 💳 Transaction Failure Analysis Dashboard

An interactive analytics dashboard for identifying, visualizing, and predicting digital payment transaction failures using Python, Flask, and data analytics. The application provides actionable insights into transaction trends, failure patterns, banking performance, merchant analytics, and customer behavior through an intuitive web interface.

---

## 📌 Overview

Digital payment systems generate millions of transactions daily, making it difficult to manually identify patterns behind transaction failures. This project automates transaction analysis by providing interactive dashboards, statistical insights, and predictive analytics to assist financial institutions in monitoring payment reliability.

---

## 🚀 Features

- 📊 Interactive analytics dashboard
- 💳 Transaction success vs failure analysis
- 🏦 Bank-wise transaction performance
- 🛍️ Merchant-wise transaction analytics
- 👤 Customer transaction insights
- 📈 Failure trend visualization
- 🔍 Search and filter transaction records
- ⚡ Real-time analytics using Flask APIs

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AbhinavSubodh/transaction-failure-analysis.git
cd transaction-failure-analysis
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Flask server

```bash
python app.py
```

Open your browser and navigate to

```
http://127.0.0.1:5000
```
